Supported OS: Windows

Description

This basic filter sample will demonstrate how to retrieve a list of filters to queue in the image processing pipeline. The xeneth SDK is delivered with a few basic built-in filters and there is also the possibility to load custom external filters. When calling the get_filter_list function a list of filter names is returned. The items from this list are passed with the queue_filter function which returns a handle to this filter on success. We will start 3 built-in filters: Autogain, Gamma and Averaging. Once set-up and the filters are running, a procedure on how to alter the position of the filters in the stack is shown. After which we remove the filters from the stack and terminate the application.

Used XCamera members

The following table lists the functions used in this sample:

Function Description
XCamera.open Opens a connection to the camera.
XCamera.close Closes the connection to the camera.
XCamera.is_initialized Check if the connection was successfully initialized.
XCamera.get_filter_list Generates list of image filters useable in QueueFilter.
XCamera.queue_filter Queue's the given filter, if found, identified by the string filtername which was retrieved from the filter list.
XCamera.is_filter_running Checks if a filter has completed it's run cycle.
XCamera.pri_image_filter Set the filter's position in the filter stack.
XCamera.rem_image_filter Removes an image filter.
