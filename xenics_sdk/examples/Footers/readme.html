Supported OS: Windows

Description

To retrieve the footer, make sure the frameBuffer is initialized with enough room to hold the frame footer. The method `create_buffer` accepts a `include_footer` argument to instruct to create extra space for the footer. Once the connection is established, a frame can be grabbed using the `XCamera.get_frame` method, only this time we add the `XGetFrameFlags.XGF_FetchPFF` flag to the GetFrameFlags to signal the API that the frame footer should be included. There are two kinds of footers: software footer, hardware footer. The software footer is created internally in the SDK, the hardware footer is coming straight from the camera. To demonstrate the contents of these footers we print the values to console and clean the connection.

Used functions

The following table lists the methods used in this sample:

Function Description
