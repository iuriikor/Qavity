Supported OS: Windows

Description

The most basic example is to open the first detected camera, start capturing, grab a frame and close the connection to the device.
We start with the EmptyProject, which was configured in 'Setting up your project', and add the functionality.

Used XCamera members

The following table lists the functions used in this sample:

Function Description
XCamera.open Opens a connection to the camera.
XCamera.close Closes the connection to the camera.
XCamera.create_buffer Creates a frame buffer.
XCamera.is_initialized Check if the connection was successfully initialized.
XCamera.start_capture Start capturing frames.
XCamera.stop_capture Stop capturing frames.
XCamera.is_capturing Check if the device is capturing frames.
XCamera.get_frame Used to grab a raw frame from the device.
