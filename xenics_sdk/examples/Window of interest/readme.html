Supported OS: Windows

Description

The window of interest (or WOI in short) is configured by making use of the property system in the xenics.xeneth.xcamera library. It allows the user to get higher framerates by decreasing the time needed to process one frame, and thus also reducing the frame size.

When the new window is set make sure that the buffer which holds the last retrieved frame is adjusted to accommodate the new frame size.
It is good practice to check the values of the properties after the WOI has changed because the given values can be altered internally in order to fit the window's constraints.
Note: Each camera has its own constraints for specifying a window. To make sure which constraints the camera has, check the camera-specific WOIxx properties. These ranges can also be retrieved using the get_property_value function.

The following properties are generic for all cameras that are capable of setting a window of interest:

Property name Description
WoiSX(0) Defines the X coordinate of the start of the window of interest.
WoiEX(0) Defines the X coordinate of the end of the window of interest.
WoiSY(0) Defines the Y coordinate of the start of the window of interest.
WoiEY(0) Defines the Y coordinate of the end of the window of interest.

To check if the camera supports window of interest retrieve the property "CapWoiCount", if this property exists the has_property function returns True.

Used functions

The following table lists the functions used in this sample:

Function Description
XCamera.open Opens a connection to the camera specified and returns a handle pointing to the camera. The handle is used in all other functions so keep a reference.
XCamera.close Closes and cleans up the connection to the camera.
XCamera.is_initialized Check if the connection was successfully initialized.
XCamera.start_capture Start capturing frames.
XCamera.stop_capture Stop capturing frames.
XCamera.is_capturing Check if the device is capturing frames.
XCamera.frame_size Used to determine the native frame size.
XCamera.get_property_value Retrieve a value from the camera specified by the property name.
XCamera.set_property_value Sends a value to the camera specified by the property name.
