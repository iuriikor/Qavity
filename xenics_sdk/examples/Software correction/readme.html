Supported OS: Windows

Description

This Python example demonstrates how to load a calibration pack into the Xeneth SDK for image correction. The calibration pack can be provided with the camera or created using the Xeneth calibration wizard. Refer to the Xeneth manual for guidance on creating a calibration pack. Please note that as these packs are specific to each camera type, one is not provided with this example. Instead, you should place your own pack in the root folder of this example and name it "pack.xca" or adjust the 'packname' variable in the main function of the Python script. The example also manually starts a SoftwareCorrection filter and retrieves its parameters.

Flag Description
XLC_StartSoftwareCorrection Starts the software correction filter after unpacking the calibration data.

Used methods

The following table lists the Python methods used in this sample:

Method Description
XCamera.open Opens a connection to the camera at the given URL.
XCamera.is_initialized Checks if the connection to the camera was successfully initialized.
XCamera.load_calibration Loads sensor calibration data from the specified file.
XCamera.queue_filter Queues a filter with the given name into the camera's filter system.
XCamera.recv_stream Receives a settings data stream from the filter with the given filter ID.
XCamera.close Closes the connection to the camera and cleans up any resources.
