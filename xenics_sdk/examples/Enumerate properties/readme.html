Supported OS: Windows

Description

In this samples all functions for retrieving en writing properties are demonstrated. The count of all properties is requested, base on this count every property is enumerated with specific details about this property.

Used members

The following table lists the members used in this sample:

Member Description
open Opens a connection to the camera specified.
is_initialized Check if the connection was successfully initialized.
close Closes and cleans up the connection to the camera.
get_property_count Gets the number of properties.
get_property_name Gets the property name.
get_property_category Gets the property category.
get_property_type Retrieves the type of a property
get_property_range Retrieves the boundaries of a property For an enumerated type this is a list of possible values, for numeric types it returns "lowvalue>highvalue".
get_property_unit Retrieve units supported by this property, default unit comes first.
get_property_value Retrieve a value from the camera specified by the property name.
XCamera.get_property_blob Retrieve a binary large object (blob) from the camera, this is usually firmware, calibration data, or overlay data.
