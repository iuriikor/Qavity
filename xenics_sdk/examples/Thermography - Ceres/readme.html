Supported OS: Windows

Description

This Python script demonstrates how to exploit the on-board radiometric data to compute the temperature of observed objects using a Ceres camera. Two thermography modes are illustrated:

  1. Temperature mode, where temperatures are calculated via linear interpolation.
  2. Radiometric mode, where temperatures can be obtained as a result of applying a dedicated analytic function.
The script assumes that a radiometric calibration is loaded in the selected image correction source on board of the camera.

Used Functions and Methods

The following table lists the functions and methods used in this script:

Function / Method Description
XCamera() Creates an instance of the XCamera class.
XCamera.open() Opens a connection to the camera specified by a URL.
XCamera.close() Closes the connection to the camera.
XCamera.is_initialized Checks if the connection was successfully initialized.
XCamera.start_capture() Starts capturing frames.
XCamera.stop_capture() Stops capturing frames.
XCamera.is_capturing Checks if the device is capturing frames.
XCamera.get_frame() Grabs a raw frame from the device.
XCamera.create_buffer() Creates a buffer for frames of a specified type.
XCamera.set_property_value() Sets the value of a specified property.
XCamera.get_property_value() Retrieves the value of a specified property.
temperature_in_temperature_mode() Calculates the temperature using linear interpolation in temperature mode.
temperature_in_radiance_mode() Calculates the temperature using a dedicated analytic function in radiance mode.
