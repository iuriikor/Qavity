Supported OS: Windows

Description

To make it easier for the user to upload these file without having to deal with buffers and file operations the `set_property_value` method of `XCamera` class can be used. The argument which is passed as the value when calling this function should contain a relative or absolute path to this file.

Used functions

The following table lists the functions used in this sample:

Function Description
XCamera.open Opens a connection to the camera specified.
XCamera.is_initialized Check if the connection was successfully initialized.
XCamera.set_property_value Sets the value of a named property.
XCamera.close Closes and cleans up the connection to the camera.
