Used functions

The following table lists the functions used in this sample:

Function Description
XCamera.open Opens a connection to the camera specified by the URL.
XCamera.create_buffer Creates a buffer with enough room to hold the frame and footer.
XCamera.is_initialized Checks if the camera connection was successfully initialized.
XCamera.colour_mode Sets the color mode of the camera.
XCamera.queue_filter Queue's the given filter.
XCamera.is_filter_running Checks if a filter is properly initialized.
XCamera.set_filter_parameter Modifies the parameter of an image.
XCamera.start_capture Starts capturing frames from the camera.
XCamera.get_frame Grabs a raw frame from the device.
XCamera.stop_capture Stops capturing frames.
XCamera.close Closes and cleans up the connection to the camera.
