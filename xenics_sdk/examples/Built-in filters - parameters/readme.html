Used Members of XCamera

The following table lists the methods and properties used in this sample:

Member Description
XCamera.open Opens a connection to the camera specified by the string parameter and returns True if successful.
XCamera.is_initialized Checks if the connection was successfully initialized.
XCamera.queue_filter Queues the given filter, if found, identified by the string filter name.
XCamera.is_filter_running Checks if a filter has completed its run cycle.
XCamera.rem_image_filter Removes an image filter.
XCamera.get_filter_parameter Gets a filter parameter.
XCamera.get_filter_parameter Sets a filter parameter.
XCamera.recv_stream Receives a stream from the camera.
XCamera.close Closes and cleans up the connection to the camera.
