Supported OS: Windows

Description

This Python example demonstrates how to save and load camera settings to and from a settings file (.xcf) using the Xeneth SDK. It shows how to change a property value and verify the saved and loaded values. The camera is initialized using a provided URL, which defaults to "cam://0?fg=none" if no argument is given.

Used functions

The following functions used in this Python example:

Function Description
XCamera.open Opens a connection to the camera specified by the provided URL.
XCamera.is_initialized Checks if the connection was successfully initialized.
XCamera.save_settings Saves the camera settings to a configuration file (settings.xcf).
XCamera.get_property_value Retrieves a property value ('IntegrationTime' in this case) from the camera.
XCamera.set_property_value Changes a property value ('IntegrationTime' in this case) of the camera.
XCamera.load_settings Loads the camera settings from a configuration file (settings.xcf).
XCamera.close Closes the connection to the camera.
