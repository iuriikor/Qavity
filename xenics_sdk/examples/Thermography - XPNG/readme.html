Supported OS: Windows

Description

In this sample we will load a thermography calibration pack and save an XPNG image using the XCamera class. This image will contain the thermal data in the extensions section of the PNG format to convert pixel data to temperatures.

Used functions

The following table lists the functions used in this sample:

Function Description
XCamera.open_camera Opens a connection to the camera specified and returns a handle pointing to the camera. The handle is used in all other functions so keep a reference.
XCamera.close_camera Closes and cleans up the connection to the camera specified by the handle which was obtained by calling the open_camera function.
XCamera.load_calibration Loads the given calibration pack, XLC_StartSoftwareCorrection will start the internal correction filter on success.
XCamera.queue_filter For thermography to work the filter has to be started as well.
XCamera.is_initialised Check if the connection was successfully initialized.
XCamera.start_capture Start capturing frames.
XCamera.stop_capture Stop capturing frames.
XCamera.is_capturing Check if the device is capturing frames.
XCamera.create_buffer Creates a frame buffer.
XCamera.get_frame Used to grab a raw frame from the device.
XCamera.get_frame_size Used to determine the native frame size.
