Supported OS: Windows

Description

When working with thermally corrected cameras, it is possible to convert the digital values into temperature units. In this sample, a connection is made, and the thermal calibration pack is loaded. The Thermography filter is initialized, with "celsius" provided as an argument to define the unit in which the temperature should be represented. Other possible units are "kelvin" and "fahrenheit".

For better performance when converting the digital values to temperature, a lookup table is prepared prior to the acquisition of images.

Used methods

The following table lists the methods used in this sample:

Method Description
XCamera.open() Opens a connection to the camera specified by the URL.
XCamera.close() Closes the connection to the camera.
XCamera.is_initialized Check if the connection was successfully initialized.
XCamera.start_capture() Start capturing frames.
XCamera.create_buffer Creates a frame buffer.
XCamera.stop_capture() Stop capturing frames.
XCamera.is_capturing Check if the device is capturing frames.
XCamera.get_frame() Used to grab a raw frame from the device.
XCamera.load_calibration() Loads sensor calibration data.
XCamera.queue_filter() Queues the filter based on the supplied name into the filter system.
XCamera.adu_to_temperature() Convert the supplied ADU value to a temperature unit.
